# Total.js Employees API in Practice

Practicing Total.js 

## Installation

1. Follow the installation guide in MySQL test db repo.
https://github.com/datacharmer/test_db

2. Clone this repo.

3. Start the server
```bash
node debug.js
```

## Usage

```bash
# GET, POST
/api/v1/employees

# PUT, DELETE
/api/v1/employees/{id}
```
